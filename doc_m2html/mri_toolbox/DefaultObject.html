SYNOPSIS

function O = DefaultObject ( Name )

DESCRIPTION 

 O = DefaultObject ( Name )
 Create an object for an AVWObjectMap.  Creates a color
 pseudo-randomly using the Name as a seed, so that the same name
 will always give the same color.
