mri_toolbox

PURPOSE

mri_toolbox - Graphical user interface (GUI) to various MRI tools

SYNOPSIS

function [ varargout ] = mri_toolbox(command)

DESCRIPTION 

 mri_toolbox - Graphical user interface (GUI) to various MRI tools
 
 The main gui is the primary store for general parameters 
 and provides access to other tools.
